# LuxCoder v0.1.99 更新

相对 `v0.1.98`：新增 NUWA API 供应商、Ink 单墨水分层视觉系统、4 个 UI/UX 修复、模型配置空校验。

## 新功能

- **NUWA API 供应商** — 新增 `nuwa` 供应商类型（OpenAI 兼容聚合平台），Base URL `https://api.nuwaapi.com/v1`，支持 Claude / GPT / DeepSeek / Gemini / Grok 等 58 个模型。
- **模型配置空校验** — 新建渠道时，若未启用任何模型，「创建」按钮和退出弹窗的「保存并关闭」均禁用，避免创建空配置。

## 视觉重构

- **Ink 单墨水分层系统** — 对齐 Cursor/Codex 的 Apple 式安静视觉：统一背景、边框、文字的色彩层级，简化全局 CSS，减少硬编码色值。

## Bug 修复

1. **CJK 文字逐字断行** — `UserMessageContent` 气泡添加 `min-w-[4em] + whitespace-normal`，修复 inline-block 宽度塌缩。
2. **AI 答案被折叠** — 过滤 `TaskCreate / TaskUpdate` 工具块，防止排在数组末尾导致最终答案被裹进「执行过程」自动折叠。
3. **思考深度弹窗冗余文案** — 移除「对齐 craft」提示。
4. **通用设置冗余文案** — 移除「对齐 craft」提示。

## 下载

- **macOS Apple Silicon** — `LuxCoder-0.1.99-arm64.dmg` / `LuxCoder-0.1.99-arm64-mac.zip`
- **Windows** — `LuxCoder-Setup-0.1.99.exe`

## macOS 打开说明（重要）

当前 Release **未配置 Apple Developer ID 签名/公证**。从浏览器下载后，macOS Gatekeeper 可能误报：

> "LuxCoder" is damaged and can't be opened.

**这不是安装包损坏。** 安装到 Applications 后执行：

```bash
xattr -cr /Applications/LuxCoder.app
open /Applications/LuxCoder.app
```
