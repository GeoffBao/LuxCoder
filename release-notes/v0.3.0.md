# LuxCoder v0.3.0 更新

相对 `v0.2.0`：Claude Pro/Max OAuth 登录、Agent 创建看板任务、思考深度滑动效果。

## 新功能

### Claude Pro/Max OAuth 登录
- **Claude OAuth 登录通道** — 新增 `anthropic-oauth` 供应商类型，支持 Claude Pro/Max 订阅用户通过 OAuth 登录使用。
- **内嵌 claude binary** — 自动生成 OAuth token，路由到 `CLAUDE_CODE_OAUTH_TOKEN` 环境变量。
- **ChannelForm 登录 UI** — 通道表单集成 Claude OAuth 登录按钮，Agent/Chat 模式分别正确处理。
- **Token 安全** — 诊断信息中自动脱敏 OAuth token，避免日志泄露。
- **Kimi K3 修复** — 修正 K3 模型 ID 从 `k3` 为 `kimi-k3`。

### Agent 创建看板任务
- **`create_task` Agent 工具** — Agent 可直接在 Work 模式下创建 Kanban 任务，含标题/描述/验收标准。
- **MCP 注册** — `create_task` 注册为内置 MCP 工具，归类于 `task` 类别。
- **Slugify 提升到 shared** — 任务 slug 生成逻辑从 kanban 移到 `@luxcoder/shared`，slug 碰撞保护跨路径共享。

### 思考深度滑动效果
- **`ThinkingLevelSlider` 组件** — 连续跟手滑动 + 松手吸附，渐变轨道颜色随深度从灰→蓝→靛→紫→粉红过渡。
- **输入栏思考弹窗统一** — AgentView Hover Popover 替换为新滑块。
- **设置页统一交互** — 「新会话默认思考深度」从 `<Select>` 下拉改为滑块，两处交互一致。
- **对齐 OpenCode / Claude Code** — 连续步进（step=0.001）、松手吸附、档位圆点逐步亮起。

## macOS 打开说明（重要）

当前 Release **未配置 Apple Developer ID 签名/公证**。从浏览器下载后，macOS Gatekeeper 可能误报：

> "LuxCoder" is damaged and can't be opened.

**这不是安装包损坏。** 安装到 Applications 后执行：

```bash
xattr -cr /Applications/LuxCoder.app
open /Applications/LuxCoder.app
```
