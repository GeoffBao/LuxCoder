# LuxCoder v0.2.0 更新

相对 `v0.1.99`：左侧栏 Sessions｜Projects 双 Tab 重构、项目直通看板、全局搜索支持项目、多项 UI 优化。

## 新功能

- **Sessions｜Projects 双 Tab 侧栏** — Agent 模式左侧栏新增 Hermes 风格双 Tab 页签，会话和项目导航分离，紧贴内容区。
- **项目直通看板** — 点击任意项目直接进入看板视图，不再经过独立的项目详情页。
- **ProjectSettingsDialog** — 看板顶部 ⚙️ 按钮弹出设置对话框，编辑项目名称、强调色、描述、默认专家、自定义 Kanban 列和 MEMORY.md。
- **项目删除功能** — Projects Tab 展开面板新增「删除」按钮，二次确认后删除项目及其资产。
- **全局搜索支持项目** — `⇧⌘F` 搜索现在同时匹配项目名称，结果直接导航到对应看板。

## UI 重构

- **移除独立项目中心** — `ProjectsHubView` 全屏视图已删除，项目管理完全收敛到左侧栏 Projects Tab。
- **移除项目详情页** — `ProjectInfoPage` 已删除，编辑功能移入 ProjectSettingsDialog，概览信息在看板中即见即得。
- **移除「会话｜看板」切换条** — `CodeMainViewSwitcher` 不再在主区域顶部显示，导航由左侧栏 Tab 承担。
- **移除 ProjectSwitcher** — 行内项目下拉控件已删除，替换为 Projects Tab。

## 视觉修复

- 置顶区与普通会话列表之间新增分隔线（Chat / Agent 模式均生效）。
- 侧栏 padding 对齐修复（会话列表与 Tab 区域左右边距一致）。
- Sessions｜Projects Tab 移至「Agent 专家」下方，紧贴内容区。

## Bug 修复

- 全局搜索快捷键 tooltip 显示正确（`⇧⌘F`，非 `⌘K`）。

## 下载

- **macOS Apple Silicon** — `LuxCoder-0.2.0-arm64.dmg` / `LuxCoder-0.2.0-arm64-mac.zip`
- **Windows** — `LuxCoder-Setup-0.2.0.exe`

## macOS 打开说明（重要）

当前 Release **未配置 Apple Developer ID 签名/公证**。从浏览器下载后，macOS Gatekeeper 可能误报：

> "LuxCoder" is damaged and can't be opened.

**这不是安装包损坏。** 安装到 Applications 后执行：

```bash
xattr -cr /Applications/LuxCoder.app
open /Applications/LuxCoder.app
```
