本地工程 → Agent 对话：可以做到“一次打开，直接开工”

关键是把“打开文件夹”解释为：查找或创建对应 Project，然后建立绑定该 Project 的 Session。用户不需要先理解 Agent 环境。

1

打开文件夹

首页、侧栏或新会话菜单中选择本地工程目录。

2

识别项目

用规范化路径查找已有 Project；没有则用文件夹名称自动创建。

3

继承上下文

环境提供 Skills/MCP，项目提供 cwd、Memory、资料与默认专家。

4

开始对话

自动新建项目 Session，Agent 直接在工程目录中读取、编辑与运行。
